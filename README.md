# 강의용 타이머 프로그램

Electron 기반의 간단한 강의용 타이머 프로그램입니다.

## 기능

- **현재 시간 표시**: 실시간으로 HH:MM:SS 형식으로 현재 시간을 표시합니다.
- **자동 상태 변경**:
  - 🌅 **수업 시작 전** (오전 9시 이전): 보라색 그라데이션 배경
  - 📚 **수업 중** (정각 ~ 49분 49초): 파란색 그라데이션 배경
  - ☕ **쉬는 시간** (매 시 50분 ~ 59분 59초): 핑크/노란색 그라데이션 배경
  - 🌙 **수업 종료** (오후 6시 이후): 남색 그라데이션 배경
- **항상 최상단 표시**: 다른 창 위에 항상 표시되어 강의 중에도 쉽게 시간 확인 가능

## 설치 및 실행

### 개발 모드로 실행

```bash
npm start
```

### EXE 파일 빌드

```bash
npm run build:win
```

빌드 완료 후 `dist` 폴더에 설치 파일이 생성됩니다.

## 시스템 요구사항

- Node.js 22.14.0 이상
- npm 10.9.2 이상
- Windows 10/11
